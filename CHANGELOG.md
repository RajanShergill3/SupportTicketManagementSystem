## [1.0.0] - 2026-07-20

- Express 5 REST API using TypeScript
- MongoDB integration with Mongoose
- Layered architecture (Controllers, Services, Repositories)
- User management APIs
- Ticket CRUD APIs
- Comment management APIs
- Ticket status workflow with controlled state transitions
- Request validation and centralized error handling
- Health check endpoint

- React 19 application with TypeScript
- Vite development environment
- Tailwind CSS styling
- Ticket management interface
- User management interface
- Dashboard page
- Login page (UI only)
- Search and filtering
- Ticket detail and comment views

- Backend integration tests using Jest and Supertest
